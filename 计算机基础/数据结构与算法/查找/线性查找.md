   

# 线性查找思想

数组中的每一个元素都与查找值对比，如果找到返回true，没有则返回false

```java
 public static int seqSearch(int[] arr, int val) {
        //线性查找是逐一比对，发现有相同值，就返回下标
        for (int i = 0; i < arr.length; i++) {
            if (arr[i] == val) {
                return i;
            }
        }
        return -1;
    }
```
